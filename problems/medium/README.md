# Medium Problems

Focus area: non-trivial algorithms inside OOP models, stronger state machines, and minor concurrency.

11. [Splitwise](splitwise/README.md): Debt simplification using heaps or graph-based balancing.
12. [Elevator System](elevator-system/README.md): Directional sweep logic using SCAN or LOOK, never FIFO.
13. [Chess](chess/README.md): Command Pattern for moves with full Undo and Redo support.
14. [BookMyShow](bookmyshow/README.md): Seat locking with timeout-based background reversion to AVAILABLE.
15. [ATM Machine](atm-machine/README.md): Chain of Responsibility for bill dispensing.
16. [Food Delivery](food-delivery/README.md): Five-state order machine with Observer-driven GPS updates.
17. [Car Rental](car-rental/README.md): Date-range availability queries instead of a single booked flag.
18. [Snake and Ladders](snake-and-ladders/README.md): Strategy for dice behavior with valid board initialization.
19. [Issue Tracker](issue-tracker/README.md): Per-issue workflow validation with dynamic transition rules.
20. [Hotel Booking](hotel-booking/README.md): Strategy-based pricing with weekend and holiday surges.
